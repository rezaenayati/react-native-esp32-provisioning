# react-native-esp32-provisioning
Package for Wifi provsioning ESP32 devices
## Installation

```sh
npm install react-native-esp32-provisioning
```

## Usage

```js
import { multiply } from "react-native-esp32-provisioning";

// ...

const result = await multiply(3, 7);
```
